# Hierarchy-table

## How to use 
```
yarn install
yarn start
```
and open http://localhost:3000/

To build production bundle:
```
yarn build
```

To run test cases:
```
yarn test
yarn test:coverage
```
## Used technologies
- React, Redux, redux-actions, reselect,
- Typescript
- Jest, Enzyme
- Babel
- Eslint
- Webpack
